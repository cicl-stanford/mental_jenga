Instructions

In this experiment your task is to judge which of the red bricks would fall off the table if the black brick wasn't there (see Figures below).


An example of a pile of bricks you may see in the experiment.

The selected white bricks are the ones that the person thought would fall off the table if the black brick wasn't there.


You will first get some experience with the physical properties of the bricks by seeing them fall on the table.

You will then see 43 different piles of bricks. For each pile, please select the red bricks that you think would fall off the table if the black brick hadn't been there. Note that each pile is stable. That is, none of the red bricks would fall off the table if the black brick remained where it is.

There will be catch trials throughout the experiment, and a check question at the end to make sure that you've paid attention. Your HIT will be rejected if you failed on the catch trials or answered the check question incorrectly.
